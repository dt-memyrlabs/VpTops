# VpTops — AI Talent Operations Agent

An AI agent that learns talent operations by ingesting organizational data. Feed it your hiring data, performance data, and candidate profiles — it builds the skills to run your hiring pipeline.

## What It Does

7 learning modules, each builds competency from YOUR data:

| Module | Function |
|--------|----------|
| T1 | Extracts role dimensions from JDs and business needs |
| T2 | Learns what makes effective job ads from your data |
| T3 | Designs assessments that predict actual job performance |
| T4 | Builds compliance constraint graph (EEOC, ADEA, W-2/1099) |
| T5 | Learns your documentation patterns, generates at scale |
| T6 | Uses post-hire performance to improve everything above |
| T7 | Matches LinkedIn profiles against learned success patterns |

**Phase 2:** Level System (T8) + Performance Prediction (T9)

## How It Works

1. Feed it data → it learns patterns
2. You validate what it learns → patterns lock
3. It produces outputs from learned patterns → you approve
4. Post-hire data feeds back → system self-improves
5. Profile matching improves with every validated known hire

## Built With

- EOS (Enlightened Operating System) — extraction engine
- Claude Code — execution environment
- Notion + Pieces — persistence layer
- Google Drive / Chrome — data ingestion
